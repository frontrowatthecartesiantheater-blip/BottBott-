02 · Content

Custom Autoblog Automation

A fully custom, end-to-end content pipeline built from scratch. Gregg records a voice memo → system transcribes it, writes a full SEO blog post in his voice, and publishes it automatically every Wednesday via cron job.

Per-Post Output
  • Blog post with topic-specific image and alt text
  • SEO metadata + JSON-LD schema
  • Internal and external link injection (EEAT + ranking signal)
  • LinkedIn caption + Facebook post drafted to a Google Sheet with the live URL, ready to share
Infrastructure
  • 58 topics seeded in the content library (100 planned) with 1 image/post
  • Automated email reminder ladder — prompts Gregg ahead of each weekly recording window
  • Full documentation + one-page quick reference delivered
Running cost: ~$2–3/month (Anthropic API only — zero hosting or tooling fees)
